#Weather International

Eu desenvolvi este projeto com o objetivo de praticar integração com APIs externas e manipulação de dados em tempo real no front-end. Trata-se de uma aplicação web simples de previsão do tempo, onde o usuário pode consultar as condições climáticas de qualquer cidade digitando o nome no campo de busca.

A aplicação consome dados da API do OpenWeather, realizando requisições HTTP assíncronas com JavaScript para obter informações atualizadas como temperatura, umidade, velocidade do vento e descrição do clima. Esses dados são processados e renderizados dinamicamente na interface, proporcionando uma experiência interativa e responsiva.

No desenvolvimento, foquei em estruturar bem o fluxo da aplicação: captura do input do usuário, chamada da API, tratamento da resposta (JSON) e atualização do DOM. Também implementei validações básicas para lidar com entradas inválidas e possíveis erros de requisição.

A interface foi construída com HTML e estilizada com CSS, buscando uma apresentação limpa e funcional. Já o JavaScript foi responsável por toda a lógica da aplicação, incluindo consumo da API, manipulação de eventos e atualização dos elementos na tela.

Esse projeto foi essencial para consolidar conceitos como:

* Consumo de APIs REST
* Programação assíncrona (fetch/async/await)
* Manipulação do DOM
* Tratamento de erros
* Organização de código no front-end

## Tecnologias e Ferramentas utilizadas

* HTML5
* CSS3
* JavaScript
* API OpenWeather
